- 웹사이트 url / (챗봇일 경우)텔레그램 등 메신저 주소
- 영화 리뷰를 작성하고 관련된 정보를 불러와 저장할수 있습니다. 저장된 정보를 바탕으로 나의 영화 선호 취향을 분석할 수 있습니다.

## 핵심 기능  Key Feature
- 영화에 대한 리뷰를 작성하고 저장합니다.
- 영화에 관련된 정보를 스크래핑으로 불러옵니다. 
- 작성한 리뷰와 불러온 정보를 기반으로 분석된 결과를 보여줍니다.
